About AI Powered Mental Stress Detector

Purpose of this Application

The AI Powered Mental Stress Detector is designed to analyze a person's mental stress level based on their typing behaviour. When users type a paragraph on the detector page, the system observes their typing patterns and evaluates certain typing characteristics. These characteristics are used to estimate whether the user is relaxed, moderately stressed, or highly stressed.

How the System Works

The system uses keystroke dynamics to analyze typing behaviour. When a user types a given paragraph, the system records several important features such as typing speed, typing accuracy, backspace usage and key hold time.

  • The user enters the detector page and a paragraph is displayed.
  • The user types the same paragraph in the typing box.
  • The system records keystroke events while typing.
  • Important typing metrics are calculated.

Typing Features Analyzed

  • Typing Accuracy: Compares typed text with the original paragraph.
  • Typing Speed: Measures how fast the user types words per minute.
  • Backspace Count: Counts how many times the user presses the backspace key.
  • Key Hold Time: Measures the duration between key press and key release.

Stress Detection Logic

Based on the typing behaviour, the system evaluates stress levels. For example, if the user presses the backspace key many times or types slowly with many errors, it may indicate higher levels of stress.

  • Backspaces less than 3 → Low Stress
  • Backspaces between 4 and 7 → Medium Stress
  • Backspaces greater than 7 → High Stress

Result and Recommendations

After the typing analysis is completed, the system displays the final result including typing accuracy, typing speed, number of backspaces and the predicted stress level. The system also provides helpful suggestions to reduce stress such as breathing exercises, eye relaxation techniques and calming music to improve mental wellbeing.
